# 5-8 分钟演示脚本

1. 打开系统入口，展示“橙意外卖”四端入口。
2. 选择用户端登录，进入首页，说明手机端 H5 风格和 Stitch 原型视觉。
3. 点击搜索或商家卡片，进入商家详情，展示菜品列表和加入购物车。
4. 进入购物车，点击去结算，展示确认订单页面。
5. 提交订单并进入模拟支付，选择模拟支付方式并支付成功。
6. 查看配送跟踪，说明订单状态会通过接口和 WebSocket 推送。
7. 切换商家端，展示工作台、订单管理、菜品管理，模拟商家接单和出餐。
8. 切换骑手端，展示接单大厅、配送任务、收入统计，模拟接单、取餐和送达。
9. 回到用户端提交评价。
10. 切换后台管理端，展示数据看板、商家审核、骑手审核和营销配置。
